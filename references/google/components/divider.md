# Divider — Google Material Design 3

## Dimensions

| Property | Value |
|----------|-------|
| Height | 1px |
| Color | `--md-sys-color-outline-variant` |

## Variants

- **Full-width**: Spans entire container
- **Inset**: 16dp padding from start (used in lists to align with text content past icons)

## Usage

- Separate list items
- Separate content sections
- Used within `<md-list>` between items
- Purely visual — no interactive behavior
