# Context Menu — Apple Human Interface Guidelines

## Trigger

| Platform | Gesture |
|----------|---------|
| iOS/iPadOS | Long press (haptic touch) |
| macOS | Right-click or Control-click |
| visionOS | Long look or pinch-and-hold |

## Visual Style

- System-styled, adapts to platform automatically
- Blurred background material
- SF Symbols for menu item icons
- Dividers for logical grouping
- Destructive items in system red

## Content

- Contextual actions relevant to the touched/clicked element
- Grouped by related function with dividers
- Icons from SF Symbols (optional, leading position)
- Submenus for nested options (disclosure indicator)
- Destructive actions at bottom, in red

## Rules

1. Keep menus concise — only most relevant actions
2. Use dividers to group related items
3. Destructive items always at bottom
4. Provide keyboard shortcuts for power users (macOS)
5. Don't use as primary navigation — only contextual shortcuts
6. Preview content above menu when applicable (iOS peek)
